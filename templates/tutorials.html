Using the Detection Tool

Follow this step-by-step guide to run aerial object detection

This page explains how to use the tool. For understanding how detection works, visit the Learn page.
1

Upload Image

Upload a drone or satellite image. Supported formats include JPG, PNG and TIFF. Higher resolution images provide better detection accuracy.

Tip: Clear aerial images with visible objects work best.
2

Adjust Confidence Threshold

The confidence slider controls how certain the model must be before showing detections.

  • Low value → more detections (may include false positives)
  • High value → fewer but accurate detections
3

Select Detection Mode

Choose between standard detection and orientation-aware detection.

  • Standard: Horizontal boxes
  • Orientation-Aware: Rotated boxes (recommended for aerial images)
4

Run Detection

Click the Run Detection button. The model processes the image and predicts objects.

Processing time depends on image resolution and GPU availability.
5

Analyze Results

The system displays input vs output images along with performance metrics.

  • Detected objects
  • Inference time
  • FPS
  • Density
6

Interpret Metrics

The performance table shows:

  • Objects: number of detected items
  • Density: objects per megapixel
  • Time: inference speed
  • FPS: processing performance
Higher FPS and lower time indicate faster detection.
