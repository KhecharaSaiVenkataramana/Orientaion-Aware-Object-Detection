About the Project

Orientation-Aware Aerial Object Detection Platform

Overview

This project is an AI-powered web platform designed to detect objects in aerial and satellite imagery. Traditional object detection struggles when objects appear rotated. This system improves detection accuracy by predicting object orientation along with location.

Problem Statement

In drone and satellite images, objects such as vehicles, ships and buildings appear at arbitrary angles. Horizontal bounding boxes capture large background areas which reduces confidence and localization accuracy.

Our Solution

We use Orientation-Aware YOLOv8 which predicts an additional rotation angle (θ) for every detected object. This allows tight bounding boxes and improves detection in dense aerial scenes.

System Workflow

The system processes aerial images and outputs oriented bounding boxes along with performance metrics such as detection time and FPS.

Key Features

  • Rotated bounding box detection
  • Confidence threshold control
  • Performance analytics (FPS, latency, density)
  • Input vs Output comparison
  • Web-based interactive interface

What Makes It Different?

  • Predicts orientation angle of objects
  • Uses rotated IoU for accurate localization
  • Designed specifically for aerial imagery
